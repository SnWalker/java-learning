# Java Learning & Practice

存放学习 Java SE 过程中的练习代码、笔记以及练手的小项目。
记录了从 Java 基础语法到核心 API，再到 Java 8 特性和综合实战的完整路径。

## 📂 目录结构

```text
java-learning/
├── java-basic/          # Java 基础语法 (变量、控制流、OOP基础)
├── java-api/            # Java 核心 API (集合框架、IO流、反射、异常处理)
├── java8/               # Java 8+ 新特性 (Lambda, Stream, Optional)
├── small-projects/      # 综合实战小项目 (应用所学知识解决实际问题)
└── notes/               # 学习笔记与技术总结
```

## 💻 练习内容说明

### 1. 基础与进阶 (Basic & API)

- **集合框架**：熟悉 List/Set/Map 的使用场景，对比 ArrayList 与 LinkedList、HashMap 的底层区别。
- **IO / NIO**：练习文件读写操作，理解字节流与字符流的区别，以及递归文件处理。
- **异常与反射**：掌握 Java 异常体系处理，初步了解反射机制在框架中的作用。

### 2. Java 8 特性

- **Lambda & Stream**：改变传统循环写法，练习函数式编程思想，提升代码简洁度。
- **Optional**：学习如何优雅地处理空指针（NPE）问题。
